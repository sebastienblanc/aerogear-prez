Encryption

garder vos données sur votre portable sécurisée

Encryption - les algos

garder vos données sécurisées sur votre portable

Les différents algos supportés:

  • PBKDF2: dérivation de key
  • Galois/Counter Mode(GCM): private key ou symmetric encryption
  • Cipher Block Chaining (CBC): private key ou symmetric encryption
  • Elliptic Curve cryptography: public/private key, asymmetric

L'encryption en JavaScript


  var rawPassword = AeroGear.Crypto().deriveKey( PASSWORD ),
     utf8String = sjcl.codec.utf8String,
     hex = sjcl.codec.hex,
     cipherText,
     options = {
         IV: hex.toBits( BOB_IV ),
         AAD: hex.toBits( BOB_AAD ),
         key: rawPassword,
         data: utf8String.toBits( PLAIN_TEXT )
      };

  cipherText = AeroGear.Crypto().encrypt( options );
  options.data = cipherText;
  plainText = AeroGear.Crypto().decrypt ( options );

L'encryption pour Android


     CryptoBox cryptoBox = new CryptoBox(new PrivateKey(BOB_SECRET_KEY));

L'encryption pour iOS


 // Generate key from pasword
 AGPBKDF2* derivator = [[AGPBKDF2 alloc] init]

Les Notifications Push

  • Message court pour notifier!
  • Pas de connection à maintenir:
    • Le terminal s'enregistre auprès de l'UPS
    • UPS se charge de contacter le réseau de Push
  • API Cliente Unifiée pour s'enregistrer et lire les messages

Apple Push Notification Service

  • iOS et Safari
  • Production/Development environment
  • Mise en place complexe
    • Provisioning Profile(s)
    • SSL Certificate(s)
  • Terminal identifié par le deviceToken
  • APIs Cliente facile d'accès

Google Cloud Messaging

  • (Google) Android and Chrome
  • Mise en place simple (SenderID + Google API Key)
  • Terminal identifié par le registrationID
  • Partie client plus complexe:
    • Beaucoup (trop!) d'XML : Manifestes
    • Pas mal d'interface á implementer

Mozilla SimplePush

  • FirefoxOS et Firefox Desktop
  • Signalisation/Notification (pas de transport de message)
  • PUB / SUB
  • Subscriptions identifié par le channelID
  • Mozilla "spec" car la W3C Push-API craint
    • Protocole définit (Ref. Server: written in Go)
    • API JavaScript définit
  • AeroGear Quickstart:
  • https://github.com/aerogear/aerogear-simplepush-quickstart/
