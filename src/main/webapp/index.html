Encryption

Keep your data secured.

Encryption - Algorithms

Keep your data secured on your device

Supported algorithms:

  • PBKDF2: Key Derivation
  • Galois/Counter Model (GCM): private key or symmetric encryption
  • Cipher Block Chaining (CBC): private key or symmetric encryption
  • Elliptic Curve cryptography: public/private key, asymmetric

Encryption with JavaScript


  var rawPassword = AeroGear.Crypto().deriveKey( PASSWORD ),
     utf8String = sjcl.codec.utf8String,
     hex = sjcl.codec.hex,
     cipherText,
     options = {
         IV: hex.toBits( BOB_IV ),
         AAD: hex.toBits( BOB_AAD ),
         key: rawPassword,
         data: utf8String.toBits( PLAIN_TEXT )
      };

  cipherText = AeroGear.Crypto().encrypt( options );
  options.data = cipherText;
  plainText = AeroGear.Crypto().decrypt ( options );

Encryption with Android


     CryptoBox cryptoBox = new CryptoBox(new PrivateKey(BOB_SECRET_KEY));

Encryption with iOS


 // Generate key from pasword
 AGPBKDF2* derivator = [[AGPBKDF2 alloc] init]

3: Cross-platform Features

UnifiedPush Server

Without UnifiedPush

With UnifiedPush

Push notifications

  • Short Messages to ... notify!
  • No connection to maintain :
    • Device registers to the UPS
    • UPS manage the rest
  • Unified Client Libraries to register and handle messages

Apple Push Notification Service

  • iOS and Safari
  • Production/Development environment
  • Complex setup
    • Provisioning Profile(s)
    • SSL Certificate(s)
  • Device identified by the deviceToken
  • Easy access Client API

Google Cloud Messaging

  • (Google) Android and Chrome
  • Easy setup (SenderID + Google API Key)
  • Device identified by the registrationID
  • Client setup more "complex":
    • A lot of (too much !) XML : Manifest files
    • Many interfaces to implement

Mozilla SimplePush

  • FirefoxOS and Firefox Desktop
  • Signalisation/Notification (no payload)
  • PUB / SUB
  • Subscriptions identified by the channelID
  • Mozilla "spec" because the W3C Push-API "sucks"
    • Protocole defined (Ref. Server: written in Go)
    • API JavaScript defined
  • AeroGear Quickstart:
  • https://github.com/aerogear/aerogear-simplepush-quickstart/
